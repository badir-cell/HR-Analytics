# 📊 HR Analytics Dashboard (Tableau)

## 📌 Project Overview  
This project analyzes employee data to uncover insights into workforce trends, employee attrition, and organizational performance. The goal is to help HR professionals make data-driven decisions related to employee retention, engagement, and recruitment.

The project includes **data cleaning in Excel** and **interactive dashboard development in Tableau** to present key HR metrics and patterns.

## 📂 Data Source  
The dataset was collected from an online source and contains employee-level information such as:
- Demographics  
- Job roles  
- Job satisfaction  
- Attrition status  

---

## 🛠️ Tools Used  
- **Microsoft Excel** – Data cleaning and preprocessing  
- **Tableau** – Data visualization and dashboard creation  

---

## 🔄 Data Processing Steps  

### 1. Data Exploration  
- Reviewed dataset structure, column names, and data types  
- Identified missing values and inconsistencies  

### 2. Data Cleaning (Excel)  
- Removed duplicate records  
- Handled missing/null values  
- Standardized column formatting  
- Cleaned text (trim spaces, corrected spelling, unified case)  
- Applied Find & Replace for consistency  

---

## 📊 Tableau Dashboard  

The dashboard includes the following components:

- **KPI Summary**  
  - Employee Count  
  - Attrition Count  
  - Attrition Rate  
  - Active Employees  
  - Average Age  

- **Attrition by Gender** (Lollipop Chart)  
- **Department-wise Attrition** (Pie Chart)  
- **Employees by Age Group** (Bar Chart)  
- **Job Satisfaction Rating** (Square Chart)  
- **Education Field-wise Attrition** (Bar Chart & Donut Chart)  

---

## ✨ Key Features  

- Interactive filters (department, demographics, job role)  
- Drill-down capabilities for deeper analysis  
- Trendlines for tracking attrition patterns  
- Clear KPI indicators for quick insights  

---

## 📈 Results and Insights  

- Identified departments with high attrition rates  
- Found patterns between job satisfaction and employee turnover  
- Highlighted demographic factors affecting attrition  
- Provided insights into workforce distribution by age and education  

---

## 💼 Business Impact  

- Supports HR teams in reducing attrition  
- Helps improve employee engagement strategies  
- Enables better workforce planning  
- Provides centralized HR performance insights
